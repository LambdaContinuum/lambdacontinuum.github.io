The “Is–Ought” Gap Is Not Merely a Matter of Logical Validity

2026/01/04

I recently saw a construal of the “Is–Ought” gap in terms of mere logical validity. It was described as the inability to validly deduce a prescriptive premise from some set of descriptive premises. Viewed this way, the gap faces multiple possible counterexamples.

First, consider the moral principle ought implies can , a principle moral philosophers will commonly endorse. The principle is that if you ought to do P , then it is semantically entailed that you can do P. This seems intuitive: to say you should do something seems to prima facie indicate that it’s possible to do. So we’re given OP ⊨ ◊P; since contraposition preserves validity, ¬◊P ⊨ ¬OP. This would be a violation of the gap.

Secondly, consider the following dilemma given by A.N. Prior:

  1. Tea Drinking is common in England.

  2. Therefore, Tea Drinking is common in England or all New-Zealanders ought be shot.

This inference is valid via disjunctive introduction. Now consider the following dilemma. Either (2) is normative or descriptive. If it’s normative this inference is a violation of the gap. If it’s descriptive consider the following deduction:

  1. Tea Drinking is common in England or all New-Zealanders ought be shot.

  2. Tea Drinking is not common in England

  3. Therefore, all New-Zealanders ought be shot.

This would be a violation of the gap.1

Third, consider the trivial case of explosion. From contradictions, everything validly follows. The counter-example is clear, assume a descriptive contradiction and you can validly infer a normative conclusion. This would be a violation of the gap. Recall the is-ought gap is explicated in terms of valid inference not sound inference.

Fourth, consider a Standard Deontic Logic and recall the Obligatory operator is closed under Logical consequence.2 This is to say that if OA is true and that A entails B, that OB is true. Given that logical truths are logically entailed by everything, it follow that all logical truths are obligatory. Consider the following: q ⊨ Op ∨ ¬Op. This would be a violation of the gap.

Don’t take this as me claiming there is no gap. There is, its construction is just going to be more sophisticated than a mere reference to logical validity.

  1. This supposed violation of the gap relies on the distinction between normative and descriptive exhausting all possible choices. It may fall apart if you think a proposition could be neither normative nor descriptive. ↩︎

  2. I think such a logic fails at properly capturing normative considerations; see my article on it if you’re curious. Knowing that, you might wonder why I give this as a counterexample. It illustrates the general point that by indexing the is–ought gap to mere logical validity, the truth of the is–ought gap relies on whichever logic you think “holds.” If someone were to adopt SDL as correctly modeling normative considerations, then the is–ought gap is trivially false. If the logic we’re considering is some basic propositional logic, then the explosion counterexample demonstrates that the is–ought gap is false, etc., etc. ↩︎
