Carnaps Inductive Logic

16 May 2026   1757 words   ~9 mins

Consider the notion of confirmation, it’s standard to talk of confirmation in a manner that gestures at an “in favor of” relationship between evidence and hypotheses. Carnap’s approach to induction was intended to give a formal explication of confirmation of some hypothesis, H, given some body of E. If E is some conjunction of all available observation data, \(c(H, E)\) is the degree of confidence or belief one should assign H given that body of evidence.

Now consider a first-order language that consists of a finite number of logically independent monadic predicates, a finite number of individual constants, and standard boolean connectives. A literal in such a language is either an atomic formula or its negation. A state description in such a language is a conjunction which, for any predicate and any constant, contains exactly one literal composed of them.

Thus, a state description for any property and any object will say whether the object has the property. It’s easy to see that any sentence is logically equivalent to the disjunction of the state descriptions which entail it. So, if every object in the domain is named by a different individual constant, then the set of all state descriptions will exhaust all possible states of the domain as describable in the language. This is best illustrated by example. Consider a language with only one predicate G and only two constants a, and b. This language has four state descriptions:

  1. \(Ga \land Gb\)
  2. \(Ga \land \neg Gb\)
  3. \(\neg Ga \land Gb\)
  4. \(\neg Ga \land \neg Gb\)

A probability measure assigns probabilities to state descriptions such that the sum of the probability measures of all state descriptions is 1. As state descriptions are mutually exclusive, the probability measure of a disjunction of state descriptions is the sum of all the probability measures of all disjuncts.

Each sentence is equivalent to a disjunction of a state description, so the probability measure will cover all sentences. Given a probability measure m,

\[ c(H, E) = \frac{m(H \land E)}{m(E)} \]

So the degree to which evidence E confirms a hypothesis H is the proportion of the probability of the hypothesis and the evidence to the probability of the evidence. It’s easy to see that there’s a variety of different confirmation functions since there’s a variety of different probability measures.

So how should we define our probability measure? One way that Carnap looks at is to divide the probabilities equally among the state descriptions. If there are k available state descriptions, and n of those state descriptions entail H, the probability of H is given by \(m^{\dagger}(H) = n/k\)

In our prior example, each state description is assigned the $m^{\dagger}$ value of \(\frac{1}{4}\). So:

\[ m^{\dagger}(\neg Ga) = m^{\dagger}(3) + m^{\dagger}(4) = 1/4 + 1/4 = 1/2 \]

The degree of confirmation of \(\neg Ga\) by \(Gb\) is also \(1/2\):

\[ c^{\dagger}(\neg Ga, Gb) = \frac{m^{\dagger}(\neg Ga \land Gb)}{m^{\dagger}(Gb)} = \frac{m^{\dagger}(c)}{m^{\dagger}(a) + m^{\dagger}(c)} = \frac{1/4}{1/2} = 1/2. \]

WLOG \(m^{\dagger} (Ga) = c^{\dagger}(Ga, Gb) = 1/2\), so observing \(Gb\) has no effect on the beliefs one should assign to \(Ga\). This problem generalizes, which led Carnap to consider a different probability measure, \(m^{*}\). This measure relies on what is called a structure description. A structure description (generated from a given state description) is the set of all state descriptions that result from some state description by a permutation of individual constants. Structure descriptions can be seen as encoding information about the numerical distribution of properties among objects. So our simple language above has three structure descriptions:

  1. \(\{Ga \land Gb\}\)
  2. \(\{Ga \land \neg Gb, \neg Ga \land Gb\}\)
  3. \(\{\neg Ga \land \neg Gb\}\)

(1) encodes that all objects have property G, (2) that exactly one object has property G, and (3) that no objects have property G.

First we assign equal probabilities amongst all structure descriptions (such that they all sum up to 1). Then divide the probability each structure description equally among its members. For example, structure descriptions (1), (2), and (3) are assigned probability measure \(1/3\). (1) and (3) have exactly one state description, each of which is assigned \(1/3\). (2) has 2 state descriptions, so we assign \(1/3\) equally amongst members of (2) such that every element of (2) has probability \(1/6\).

Calculating \(c^*(\neg Ga, Gb)\):

\[ c^*(\neg Ga, Gb) = \frac{m^*(\neg Ga \land Gb)}{m^*(Gb)} = \frac{m^*(c)}{m^*(a) + m^*(c)} = \frac{1/6}{3/6} = 1/3 \]

The probability measure of \(\neg Ga\):

\[ m^*(\neg Ga) = m^*(c) + m^*(d) = 1/3 + 1/6 = 1/2 \]

Since \(c^*(\neg Ga, Gb) < m^*(\neg Ga)\), \(Gb\) disconfirms \(\neg Ga\).

Now above I said there’s a variety of confirmation functions. This undersells it. If \(F_1, F_2, \dots, F_k\) are monadic predicates of a language, we say that a Q-formula predicated of a constant a is of the form:

\[ \pm F_1 a \land \pm F_2 a \land ... \land \pm F_k a \]

where \(\pm\) stands for negation or nothing. Q-formulas of such a language can be enumerated, say you pick the i-th one and call it \(Q_i\). What’s of interest is the ability to compute how much complete information of an object confirms a given complete description of some new object. We do this by calculating \(c(H_{Q_{i}}, E_{Q})\) where \(H_{Q_{i}}\) is the \(Q_i\) formula predicated of a certain constant a and \(E_q\) is a conjunction of certain Q-formulas predicated of some constants different from a.

Carnap suggests there are at least two important factors in our assessment of \(c(H_{Q_{i}}, E_{Q})\):

  1. One is the empirical factor of the relative frequency of \(Q_i\)s in \(E_Q: s_i/s\) where \(s_i\) is the number of occurrences (modulo logical equivalences) of \(Q_i\) in E and s is the number of non-equivalent Q-formulas in E.
  2. The other is the logical one: the logical factor of \(Q_i\) equals \(1/K\), where $K$ is the number of all Q-predicates of the language.

Following Carnap, \(c(H_{Q_i}, E_Q)\) should be somewhere between these two values. A convenient way of representing this is to take it to be their weighted mean defined by:

\[ c(H_{Q_i}, E_Q) = \frac{\frac{w_1s_i}{s} + \frac{w_2}{K}}{w_1 + w_2} \]

Since what matters is the ratio of the weights, we can follow Carnap in parametrizing \(w_1\) by taking it to be \(s\), guaranteeing the empirical factor gains weight as more observations are made. Call the other weight \(\lambda\):

\[ c(H_{Q_i}, E_Q) = \frac{s_i + \lambda/K}{s + \lambda} \]

Any choice of \(\lambda\) gives us a new confirmation function. Let’s look at the extremes of our confirmation functions, maximal importance to evidence and maximal importance to the logical factor. Setting \(\lambda = 0\):

\[ c(H_{Q_i}, E_Q) = \frac{s_i + 0/K}{s + 0} = s_i/s \]

This corresponds to Reichenbach’s straight rule, in that it assigns maximal role to the evidence and zero role to logical possibility.1 This can be seen by imagining there are three objects, of which you’ve observed two. Both objects turned out to be red. We may ask what the confirmation value of the hypothesis that the last object is also red:

\[ c(Red(c), Red(a) \land Red(b)) = s_{Red}/s = 2/2 = 1 \]

Suppose instead only one object was red, the other wasn’t. That means:

\[ c(Red(c), Red(a) \land \neg Red(b)) = s_{Red}/s = 1/2 \]

What about the case of maximal importance to the logical factor? Consider the limit of \(\lambda\) as it approaches infinity:

\[ \lim_{\lambda \to \infty} c(H_{Q_i}, E_Q) = \lim_{\lambda \to \infty} \frac{s_i + \lambda/K}{s + \lambda} = \frac{1}{K} \]

It’s clear that on this confirmation function it doesn’t matter whether we observe any other red objects, the confirmation of the hypothesis that the next object will be red is just the prior probability of the hypothesis. This shows us that \(\lim_{\lambda \to \infty} c(H_{Q_i}, E_Q)\) is our previously defined confirmation function \(c^\dagger\):

\[ c^\dagger = \lim_{\lambda \to \infty} c(H_{Q_i}, E_Q) \]

Where is \(c^*\) located on our continuum? When you set \(\lambda = k\):

\[ c(H_{Q_i}, E_Q) = \frac{s_i + K/K}{s + K} = \frac{s_i + 1}{s + K} \]

Given the infinite amount of probability measures, one may ask how one determines which measure one should prefer (if any)? Some have pointed out that given certain restrictions, there are substantive claims that hold no matter what non-extreme function we choose.2 Carnap himself adopted a laissez-faire view towards the end, thinking there was a range of open options to pick from.

You may analogize it to special relativity theory: it’s not velocity that is objective, rather velocity with respect to a frame of reference. Similarly, confirmation isn’t objective, rather confirmation with respect to a probability measure.

A more serious problem is reasoning by analogy. Intuitively, the more primitive properties two objects share, the more likely it should be that they would agree on other properties. Yet \(c^*\) fails to capture this intuition. For instance, suppose we have a language with two predicates $F$ and $G$ and two constants $a$ and $b$. If reasoning by analogy worked, then the fact that $Fa \land Fb \land Ga$ should give more support to the hypothesis that $Gb$ than just the evidence $Ga$:

$$c^*(Gb, Fa \land Fb \land Ga) > c^*(Gb, Ga)$$

And yet, that isn’t true, because in this case both degrees of confirmation are equal! Carnap attempted to deal with such issues by introducing a new parameter, $\eta$, but the success is quite limited.

Another serious problem: once we generalize the notions to infinite domains, Carnap’s inductive methods a priori assign zero probabilities to universal generalizations. This is considered a problem because usually laws of nature are taken to be universal and if it were true that no finite evidence can provide support for any universal statement, this would go against our intuitions that certain scientific hypotheses are better confirmed than others.

There’s also a general problem with syntactic approaches to confirmation. Goodman’s new riddle of induction shows us it’s not trivial that all predicates that can be utilized in law-like generalizations are those that the natural sciences are purported to give. Carnapian inductive theory is inadequate, given that which predicates can be sensibly used in inductive reasoning is an extralogical issue.

Reichenbach argues we should apply the Straight Rule, which says that one should take observed relative frequencies to be the limiting relative frequencies (and adjust as new observations are made). ↩︎
